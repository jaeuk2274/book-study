### 03.함수
- 작게, 더 작게 만들어라
  - if/else/while 에 들어가는 블록은 한 줄. 거기서 함수 호출. 바깥을 함수가 작아질 뿐더러 블록 안의 호출 함수명을 적절히 지으면 코드를 이해하기 월등히 쉬움.
  - 들여쓰기 수준은 1,2단 넘어서면 안됨. 함수는 읽고 이해하기 쉬워진다.
  
- 한 가지만 해라, 한 가지를 잘 해야 한다, 그 한 가지만을 해야 한다

- 함수 당 추상화 수준은 하나로!

- 위에서 아래로 코드 읽기:내려가기 규칙
